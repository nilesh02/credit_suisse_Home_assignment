How to Run the code-

Eclipse-
1) Import the code in eclipse.
2) Verify Folder with name "Files" is there containing files where data is stored.
3) Build the code with expected jar (Junit and hamcrest-core)
4) Run the code with right-click->Run on SuspiciousAcitivity.java

Command Line-
1) Set the path for Jars in Environment Variable
2) Verify Folder with name "Files" is there containing files where data is stored.
3) Type command 'javac SuspiciousAcitivity.java'
4) Type command 'java SuspiciousAcitivity
